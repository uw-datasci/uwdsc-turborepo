### New Animation Components (PR #90)

**Advanced Animation Components**
- **Marquee**: Scrolling content component with customizable direction, speed, and pause-on-hover
  - Supports horizontal and vertical scrolling
  - Configurable repeat count and gradient overlays
- **WarpBackground**: Perspective grid with animated beams
  - Customizable beam count, size, delay, and duration
  - Dynamic grid color support
- **TypingAnimation**: Text typing effect with cursor
  - Configurable duration and cursor visibility
  - Smooth character-by-character animation
- **SpinningText**: Circular text animation around a center point
  - SVG-based with customizable radius and font size
- **ShimmerButton**: Interactive button with shimmer effect
  - Customizable shimmer color, size, and duration
  - Built-in hover and active states

**Visual Effects**
- **Meteors**: Animated meteor shower effect with customizable count and styling
- **OrbitingCircles**: Elements that orbit around a central point
  - Configurable orbit radius, speed, and direction
  - Supports reverse animation and icon sizing

**Icon Updates**
- Added `FlyingSaucerIcon` from Phosphor Icons
