# UI Package

The shared design system built with shadcn/ui components.

## Overview

The `@uwdsc/ui` package contains all shared UI components (atoms) used across the monorepo.

**Location**: `packages/ui/`

**Import**: `@uwdsc/ui`

## Available Components

### Layout Components

- **Card** - Container with border and padding
- **Separator** - Visual divider
- **Aspect Ratio** - Maintain aspect ratios

### Navigation

- **Navigation Menu** - Accessible navigation
- **Breadcrumb** - Hierarchical navigation
- **Tabs** - Tabbed interfaces

### Form Components

- **Button** - Interactive buttons with variants
- **Input** - Text input fields
- **Checkbox** - Boolean input
- **Select** - Dropdown selection
- **Form** - Form wrapper with validation

### Feedback

- **Alert** - Important messages
- **Toast** - Temporary notifications
- **Badge** - Status indicators
- **Progress** - Progress indicators

### Overlays

- **Dialog** - Modal dialogs
- **Dropdown Menu** - Contextual menus
- **Popover** - Floating content
- **Sheet** - Side panels

### Data Display

- **Avatar** - User avatars
- **Table** - Data tables
- **Accordion** - Collapsible content

### Animation Components

- **Marquee** - Scrolling content with customizable direction and speed
- **TypingAnimation** - Text typing effect with cursor
- **SpinningText** - Circular text animation
- **ShimmerButton** - Button with shimmer effect
- **Meteors** - Animated meteor shower effect
- **OrbitingCircles** - Elements orbiting around a center point
- **WarpBackground** - Perspective grid with animated beams

## Installation

### Adding New Components

Use the custom script from the root:

```bash
pnpm ui:add <component-name>
```

**Examples:**

```bash
# Add a dialog component
pnpm ui:add dialog

# Add multiple components
pnpm ui:add dropdown-menu popover sheet
```

This script:
1. Changes to `packages/ui` directory
2. Runs `shadcn@canary add <component-name>`
3. Downloads component to `src/components/`
4. Installs dependencies
5. Makes it available via `@uwdsc/ui`

### Manual Installation

If you need to install manually:

```bash
cd packages/ui
pnpm dlx shadcn@canary add <component-name>
```

## Usage

### Basic Import

```tsx
import { Button, Card, CardHeader, CardContent } from "@uwdsc/ui";

export function MyComponent() {
  return (
    <Card>
      <CardHeader>Title</CardHeader>
      <CardContent>
        <Button>Click me</Button>
      </CardContent>
    </Card>
  );
}
```

### With Variants

```tsx
import { Button } from "@uwdsc/ui";

export function ButtonExample() {
  return (
    <div className="flex gap-2">

      <Button variant="outline">Outline</Button>
      <Button variant="ghost">Ghost</Button>
      <Button variant="destructive">Destructive</Button>
    </div>
  );
}
```

### With Sizes


## Styling

### Tailwind Classes

All components accept `className` prop:

```tsx
<Button className="mt-4 w-full">
  Full Width Button
</Button>
```

### CSS Variables

Components use CSS variables for theming:

```css
:root {
  --background: 0 0% 100%;
  --foreground: 222.2 84% 4.9%;
  --primary: 222.2 47.4% 11.2%;
  --primary-foreground: 210 40% 98%;
  /* ... more variables */
}
```

### Dark Mode

Automatically supports dark mode:

```tsx
import { ThemeProvider } from "next-themes";

function App({ children }) {
  return (
    <ThemeProvider attribute="class" defaultTheme="system">
      {children}
    </ThemeProvider>
  );
}
```

### Form Component

```tsx
import { useForm } from "react-hook-form";
import { zodResolver } from "@hookform/resolvers/zod";
import { z } from "zod";
import {
  Form,
  FormField,
  FormItem,
  FormLabel,
  FormControl,
  FormMessage,
  Input,
  Button,
} from "@uwdsc/ui";

const formSchema = z.object({
  email: z.string().email(),
});

export function FormExample() {
  const form = useForm({
    resolver: zodResolver(formSchema),
  });

  return (
    <Form {...form}>
      <form onSubmit={form.handleSubmit(onSubmit)}>
        <FormField
          control={form.control}
          name="email"
          render={({ field }) => (
            <FormItem>
              <FormLabel>Email</FormLabel>
              <FormControl>
                <Input placeholder="email@example.com" {...field} />
              </FormControl>
              <FormMessage />
            </FormItem>
          )}
        />
        <Button type="submit">Submit</Button>
      </form>
    </Form>
  );
}
```

## Utilities

### cn() Function

Merge Tailwind classes:

```tsx
import { cn } from "@uwdsc/ui/lib/utils";

<div className={cn(
  "base-classes",
  isActive && "active-classes",
  className
)} />
```

### Custom Hooks

```tsx
import { useToast } from "@uwdsc/ui";

export function Component() {
  const { toast } = useToast();

  const showToast = () => {
    toast({
      title: "Success",
      description: "Action completed successfully",
    });
  };

  return <Button onClick={showToast}>Show Toast</Button>;
}
```

## Best Practices

### ✅ Do

- Import components from `@uwdsc/ui`
- Use provided variants and sizes
- Extend with Tailwind classes
- Follow accessibility guidelines
- Use semantic HTML

### ❌ Don't

- Modify component files directly
- Create duplicate components in apps
- Override core styles excessively
- Ignore accessibility features

## Customization

### Extending Components

Create custom variants:

```tsx
import { Button } from "@uwdsc/ui";
import { cn } from "@uwdsc/ui/lib/utils";

export function CustomButton({ className, ...props }) {
  return (
    <Button
      className={cn("custom-gradient shadow-lg", className)}
      {...props}
    />
  );
}
```

### Theme Customization

Modify CSS variables in `globals.css`:

```css
:root {
  --primary: 200 100% 50%;  /* Custom primary color */
}
```

## Animation Components

### Marquee

Scrolling content component with customizable direction and speed.

```tsx
import { Marquee } from "@uwdsc/ui";

export function MarqueeExample() {
  return (
    <Marquee pauseOnHover className="[--duration:20s]">
      <div className="flex gap-4">
        {items.map((item) => (
          <div key={item.id}>{item.content}</div>
        ))}
      </div>
    </Marquee>
  );
}
```

**Props:**
- `reverse` - Reverse animation direction
- `pauseOnHover` - Pause on mouse hover
- `vertical` - Vertical scrolling
- `repeat` - Number of times to repeat content

### TypingAnimation

Text typing effect with blinking cursor.

```tsx
import { TypingAnimation } from "@uwdsc/ui";

export function TypingExample() {
  return (
    <TypingAnimation
      className="text-2xl font-bold"
      duration={100}
    >
      Coming Soon!
    </TypingAnimation>
  );
}
```

**Props:**
- `duration` - Milliseconds per character
- `className` - Custom styling

### ShimmerButton

Interactive button with animated shimmer effect.

```tsx
import { ShimmerButton } from "@uwdsc/ui";

export function ShimmerExample() {
  return (
    <ShimmerButton
      shimmerColor="#ffffff"
      shimmerSize="0.05em"
      shimmerDuration="3s"
      background="rgba(0, 0, 0, 1)"
    >
      Become a Sponsor
    </ShimmerButton>
  );
}
```

**Props:**
- `shimmerColor` - Color of shimmer effect
- `shimmerSize` - Size of shimmer
- `shimmerDuration` - Animation duration
- `background` - Button background color

### WarpBackground

Perspective grid with animated light beams.

```tsx
import { WarpBackground } from "@uwdsc/ui";

export function WarpExample() {
  return (
    <WarpBackground
      perspective={500}
      beamsPerSide={8}
      beamSize={3}
      gridColor="#3b82f6"
    >
      <div className="relative z-10">
        {/* Your content here */}
      </div>
    </WarpBackground>
  );
}
```

**Props:**
- `perspective` - CSS perspective value
- `beamsPerSide` - Number of animated beams per side
- `beamSize` - Width of beam in pixels
- `beamDelayMin/Max` - Random delay range
- `beamDuration` - Animation duration in seconds
- `gridColor` - Color of grid lines

### Meteors

Animated meteor shower effect for backgrounds.

```tsx
import { Meteors } from "@uwdsc/ui";

export function MeteorExample() {
  return (
    <div className="relative h-screen overflow-hidden">
      <Meteors number={20} />

        {/* Your content here */}
      </div>
    </div>
  );
}
```

**Props:**
- `number` - Number of meteors to display

### OrbitingCircles

Elements that orbit around a central point.

```tsx
import { OrbitingCircles } from "@uwdsc/ui";

export function OrbitExample() {
  return (
    <div className="relative h-[500px] flex items-center justify-center">
      <div className="relative w-36 h-16">
        {/* Center content */}
      </div>
      <OrbitingCircles
        iconSize={80}
        radius={240}
        speed={1}
        reverse
      >
        {/* Orbiting content */}
      </OrbitingCircles>
    </div>
  );
}
```

**Props:**
- `iconSize` - Size of orbiting elements
- `radius` - Orbit radius in pixels
- `speed` - Animation speed multiplier
- `reverse` - Reverse orbit direction

### SpinningText

Circular text animation around a center point.


**Props:**
- `radius` - Circle radius
- `fontSize` - Font size in pixels
- `duration` - Rotation duration in seconds
